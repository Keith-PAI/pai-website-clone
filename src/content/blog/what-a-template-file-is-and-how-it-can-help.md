If you've ever accidentally saved over a standard document and spent time recreating it, you already understand why template files exist.

## What Is a Template File?

A template file is a pre-formatted document that serves as a starting point for new documents. In Microsoft Word, templates use the `.dotx` extension (or `.dotm` if they contain macros). When you open a template, Word creates a new document based on it — leaving the original template unchanged.

## Why Templates Matter

Templates provide:

- **Consistency** — Every document starts with the same formatting, styles, and structure
- **Efficiency** — No need to recreate standard elements for each new document
- **Protection** — The original template can't be accidentally overwritten through normal use
- **Quality** — Built-in styles ensure proper formatting throughout the document

## Creating a Template in Word

1. Create a document with your desired formatting, styles, and boilerplate content
2. Choose "Save As" and select "Word Template (*.dotx)" as the file type
3. Save it to your Templates folder or a shared location
4. To use it, double-click the template file — Word will create a new document based on it

## Best Practices

- Include placeholder text to guide users on what content goes where
- Set up styles for headings, body text, captions, and other recurring elements
- Include any standard headers, footers, logos, or boilerplate language
- Review and update templates periodically to reflect current standards
